# RepairTrack

RepairTrack es un sistema de gestión de reparaciones de celulares diseñado para talleres pequeños que necesitan controlar órdenes, estados de reparación y reportes básicos de ingresos.

## Problema que resuelve

Muchos talleres gestionan sus reparaciones en papel o WhatsApp, lo que provoca pérdida de información, desorden y dificultad para dar seguimiento a los equipos.
